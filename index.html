This application demonstrates basic login functionality. Allows register new users & validates credentials. Registered users' list are stored in-memory. List of users are displayed in the console logs, as a reference. Credentials are hashed using bcrypt, to ensure password safety.

Only new users can be added to the application. Once registered, application authenticates users with their credentials / password. If invalid, user has an option to register afresh. When application is brought down or restarted, registered users are all deleted.
